# UD2 – Práctica 2 – Navegación

Esta aplicación es una refactorización de proyectos anteriores, cuyo objetivo principal es implementar patrones de navegación anidada utilizando el sistema de enrutamiento basado en ficheros de Expo Router.

## Objetivo de la Práctica

El objetivo es gestionar la navegación de la aplicación a través de un Drawer y de Pestañas (Tabs), utilizando la notación del sistema de ficheros de Expo Router. Adicionalmente, se debe asegurar que el usuario pase por una pantalla de bienvenida inicial.
